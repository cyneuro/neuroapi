# Neuro API

## Information 

Quickly develop [NEURON](https://neuron.yale.edu) models from the web.

## Running Neuro API

### Docker Start

```
git clone https://github.com/cyneuro/neuroapi.git
cd neuroapi
sudo docker-compose up -d
```

### Manual Start

```
git clone https://github.com/cyneuro/neuroapi.git
cd neuroapi
python setup.py #develop
```
